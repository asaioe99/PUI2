# PUI2

## 環の性質
### 定義
1. 加法に関して加法群である
2. 乗法に関して結合律を満足する
3. 乗法単位元 $ \epsilon \in R$ が存在する
4. 分配律を満足する

特に可換なものは可換環という。

### 左零因子及び右零因子
環 $R$の元 $a$ について、零元 ${} 0 _R $
